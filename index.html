CONCEPTOS GENERALES

Conceptos Generales

  • Eliminar siempre los console.log para entregar un proyecto.
  • Script: Son ordenes o instrucciones que serán ejecutadas por el navegador.
  • Sentencia: Es una serie de instrucciones.
  • No se define el tipo de variables
  • Javascript tiene tipado debil
  • Las variables no precisan que se le asignen un tipo especifico de dato
  • Hay 2 formas de declarar variables, no hay que declarar el tipado
  • En JavaScript no es necesario el uso de punto y coma
  • En JavaScript tampoco se usan las eñes

Console.log

Console.log es un método que me va a permitir visualizar en consola un mensaje, este mensaje puede ser tanto una cadena de texto, como también cualquier tipo de dato, etc... 

                                
                                console.log("Asi se codea!!")

Sintaxis

Son reglas que se establecen a la hora de generar instrucciones, en JavaScript no se tienen en cuenta los espacios o los saltos de línea. 

                                
                                   [asasdasd] ---> [ asasdasd ]
                                {
                                asdasdasda: asdasdas
                                }

Como tambien identacion o sangria.

Método camelCase

Se hace el uso de mayuscula en la primera letra que deviene de una palabra anterior, la primera palabra siempre va a estar en minuscula, las consecuentes, solamente las mayusculas en la primera letra. 

                                
                                 verificarUsuario
                                const carritoDeCompras = true
                                const emailUsuario = true

Comentarios

Se pueden incluir comentarios en el codigo de Javascript, existen 2 formas de colocar comentarios: 

                                
                                // Esto es un comentario de linea.
                                /* Esto es un comentario de bloque */

Case Sensitive

JavaScript es case-sensitive osea quiere decir que es sencible a las mayusculas de minusculas. Veamos un ejemplo:

  • PERRO
  • perro
  • Perro
  • PerRo
  • PERro

Todo esto es diferente para JavaScript. 

                                
                                const PerRo = no usar!!!
                                const perro = correcto
                                const perroBonito = correcto ===> case-sensitive

Variables

Una variable es un espacio reservado en la memoria que, como su nombre indica, puede cambiar de contenido a lo largo de la ejecución de un programa. En las variables almacenamos diversos tipos de datos que utilizamos en la aplicación. 

                                
                                        //Declaración de variable ES5. 
                                        var nombre = “John”; // DEPRECADO!!!

                                        //Declaración de variable ES6.
                                        let apellido = “Doe”;
                                        const curso = "JavaScript";

Declaracion de Variables

Declarar una variable significa crearla. Para esto usamos la palabra reservada let o const. Escribimos una de estas palabras claves seguido del nombre de nuestra variable. Para los nombres no debemos utilizar ni espacios ni caracteres especiales. 

                                
                                    // Correcto
                                    let apellido
                                    let telefono
                                    const anioNacimiento
                                    
                                    // Incorrecto
                                    var año // DEPRECADO Y USA EÑE!!!
                                    var teléfono móvil // DEPRECADO Y USA ACENTOS!!!

Asignacion de Valores

En una variable podemos asignar distintos tipos de valores mediante el operador de asignación, que es el símbolo de igualdad (=). 

                                
                                    // Declaración de Variables
                                    let edad
                                    let nombre
                                    let apellido

                                    // Asignación de Valores
                                    edad = 5
                                    nombre = "John R."
                                    apellido = 'Doe'

Inicializar Variables

Podemos declarar una variable y asignarle un valor inicial en el mismo proceso: 

                                
                                    // Variables Inicializadas
                                    let edad = 5 
                                    const nombre = "John R."
                                    const apellido = 'Doe'

Variable - (Const)

Una constante const recibe una única asignación al momento de su declaración, impidiendo que su valor se modifique luego.

  • const: Es una declaracion de variable que va a generar que el valor al cual se asigna sea constante.
  • gustoFavoritoDeFernando: Es el (identificador) y es aquello que le va a dar nombre a ese elemento que esta guardando, necesario a la hora de declarar variables.
  • "Menta": Es un (valor) o un (dato) de ese elemento que estamos guardando. 
                                
                                const gustoFavoritoDeFernando = "Menta"
                                console.log(gustoFavoritoDeFernando) // Menta

                                const nombre = "Carolina"
                                console.log(nombre) // Carolina

Intentar reasignar una variable en JavaScript, que sea tipo const, nos dara error. 

                                
                                    gustoFavoritoDeFernando = "Chocolate Suizo"
                                    console.log(gustoFavoritoDeFernando) // error

Variable - (Let)

Una variable let puede recibir múltiples asignaciones en el transcurso de la aplicación, es decir que puede cambiar de valor varias veces.

  • let: Es una forma de declaracion de variable que permite la reasignacion del valor.
  • Declara una variable local en un bloque de ambito, inicializandola opcionalmente a un valor. Y podras cambiar su valor luego. 
                                
                                    //Declarando la variable
                                    let edad = 25
                                    console.log(edad) // 25

Intentar reasignar una variable en JavaScript, que sea tipo let, nos cambiara su valor inicial. 

                                
                                    // Reasignando el valor de la variable
                                    edad = 29
                                    console.log(edad) // 29

Variable - (Var)

No se usa el var para asignar variables Deprecado. 

                                
                                   var = email = miguel@gmail.com // DEPRECADO!!
Se recomienda no usar o declarar variables con VAR. Ya que esta obsoleto por JavaScript.

Datos Primitivos

Los datos primitivos son tipos de datos que no precisan instrucciones para manipular valores

String:

Son cadenas de caracteres y es el uso de comillas "", '', ``. Si intentas escribir texto sin comillas, JavaScript interpreta que estas escribiendo el identificador de una variable. 

                                
                                    const segundoNombre = "Angel"
                                    console.log(segundoNombre) // Angel

Por ejemplo: 

                                
                                "Carola"
                                `Miguel`
                                'Angel'

Number:

Todos los numeros se pueden asignar a una variable, en JavaScript los numeros con coma se llaman numeros flotantes. 

                                
                                    const numeroFavorito = 20554154
                                    const numeroConComa = 2.151515 // Asi se representan numeros con coma.

Por ejemplo: 

                                
                                2022
                                1.52
                                25468

Operadores Aritmeticos

Con variables de valores numéricos podes realizar operaciones matemáticas: sumas, restas, multiplicaciones, etc. 

                                
                                    let numeroA = 1;
                                    let numeroB = 2;
                                    const numeroC = 3;

                                    //Suma de dos números (1 + 2 = 3)
                                    let resultadoSuma = numeroA + numeroB;

                                    //Resta de dos números (2 - 1 = 1)
                                    let resultadoResta = numeroB - numeroA;

                                    //Producto de dos números (2 * 3 = 6)
                                    let resultadoProducto = numeroB * numeroC;

Suma: 

                                
                                console.log(1 + 1) // 2

Resta: 

                                
                                console.log(10 - 5) // 5

Multiplicacion: 

                                
                                console.log(5 * 10) // 50

Division: 

                                
                                console.log(10 / 2) // 5

CONCATENAR 

                                
                                const usuario1 = "Agustin"
                                const usuario2 = "Andres"
                                const usuario3 = "Joaquin"

                                console.log(usuario1 + ", " + usuario2 + " y " + usuario3) // Agustin, Andres y Joaquin

Prioridad 

                                
                                console.log(2 * 4 + 5)
                                const suma = 2 * 5 + 10 // Sin console.log no se podria ver el resultado que es 20.

Cuadros de Dialogo

Es una ventana emergente la cual interactua el usuario, es nativa del navegador. Esta venta no es customizable y solo se utiliza para aprender lo basico de JavaScript.

alert

La sentencia alert() mostrará una ventana sobre la página web que estemos accediendo mostrando el mensaje que se pase como parámetro a la llamada. En resumen sirve para visualizar un mensaje del usuario: 

                                
                                    alert("Hola bienvenidos a JuniorBookJS!")

prompt

La sentencia prompt() mostrará un cuadro de diálogo para que el usuario ingrese un dato. Se puede proporcionar un mensaje que se colocará sobre el campo de texto. El valor que devuelve es una cadena que representa lo que el usuario ingresó: 

                                
                                    const edadRealDelUsuario = prompt("Ingrese su edad")
                                    console.log(`Usted tiene: ${edadRealDelUsuario}`)

typeof

Es un metodo dice cual es el tipo de dato que le pasaste: 

                                
                                    console.log(typeof 42); // "number"
                                    
                                    console.log(typeof 'blubber'); // "string"

                                    console.log(typeof true); // boolean

                                    console.log(typeof undeclaredVariable); // "undefined"
